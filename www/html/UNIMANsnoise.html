Dataset UNIMANsnoise.

Description

The dataset contains the statistics on degradation in the individual performance of UNIMAN sensor in terms of standard deviation of sensitivity coefficients computed over the long-term UNIMAN dataset.

Details

The datasets has one variable Bsd of the class list. The sd values themselves are stored in a matrix of 3 rows and 17 columns under two categories:

class The class name: SensorModel and Sensor.
model The model name: mvr and plsr.
